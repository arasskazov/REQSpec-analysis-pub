|---|----------------------------------------------|--------------------------|-------------------|---------------|---------------------|-------------|
| 1 | Общие сведения | | | | | |

| 3 | Характеристика объектов автоматизации | | | | | |
| 4 | Требования к системе | | | | | |

| 7 | Требования к документированию | | | | | |
| 8 | Источники разработки | | | | | |

ШАГ 2: Для каждого раздела с оценкой < 3 укажи:
- Что конкретно отсутствует (со ссылкой на п. ГОСТ)

#### Блок 1.2: Актуальность нормативной базы

**Промпт**:
```
БЛОК 1.2 — ПРОВЕРКА АКТУАЛЬНОСТИ НПА

Выполни проверку нормативно-правовых актов, упомянутых в ТЗ.

ШАГ 1: Извлеки ВСЕ упоминания ГОСТов, РД, СП и других НПА из текста ТЗ. Результат в таблице:

| № | Обозначение в ТЗ | Раздел ТЗ | Статус (действует/отменён/заменён) | Актуальная версия | Критичность замены |
|---|-----------------|-----------|-----------------------------------|-------------------|-------------------|
| | ГОСТ 34.602-89 | | | ГОСТ Р 34.602-2020 | ВЫСОКАЯ |
| | | | | | |

ШАГ 2: Для каждого отменённого/замененного стандарта укажи:
- Дату отмены
- Чем заменён
- Влияет ли замена на требования ТЗ (Да/Нет/Требует уточнения)

ШАГ 3: Проверь противоречия между стандартами:
- Конфликтуют ли требования разных ГОСТов, упомянутых в ТЗ?
- Есть ли дублирование требований из разных стандартов?

ФОРМАТ ОТВЕТА: Таблица + список критичных замен с рекомендациями.
```
